# Myssm
this is a ssm project

## v1.0.0 
MYSSM框架 
整合了SPRING+SPRINGMVC+MYBATIS 
同时也加入了MyBatis Generator自动创建代码功能，并且更进一步, 
使用FreeMarker模板引擎，生成model实体，mapper接口，MapperXml，Service，ServiceImpl，以及Controller、页面js和jsp,
同时兼容mysql、mssql、oracle数据库
